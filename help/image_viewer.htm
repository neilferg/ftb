Image Viewer

Clicking on any image link will cause the image viewer to be launched. This viewer has enhancements over the browser's standard in image viewer, in particular it allows images to be zoomed (magnified) and panned. The screen shot below shows a typical image being viewed.



To use the viewer:
  • Clicking the red 'X' button closes the viewer, as does the ESC key.
  • Rolling the mouse wheel zooms in and out.
  • Holding the mouse right button down whilst dragging will pan.
