Family List

The Home page displays a list of the Families (surnames) that family trees are available for. Clicking on a family takes you to the tree diagram for that family.
NOTE: Larger tree diagrams will require you to scroll the browser window to pan around the tree.

Navigation Menu

At the top of every page is a menu-bar. Here's a brief explanation each menu item:


 Takes you back to the home page. This item is always present.
From the home page you may select a family tree diagram to browse.

Search for a person. Enter any part of the person's name in the box and click the Go! button next to it (or enter). The search results will present you with a list of individuals from which you can select.

Takes you to the help pages. This item is always present.
This menu item is available when viewing an individual person's details. It takes you to the their family tree and highlights within it.
This menu item is available when viewing an individual person's details. It takes you back to that individual's home page.

When viewing an individual person, this menu item takes you back one generation to their parent, usually this will be their father.

Don't forget that you can also use the web browser's navigation keys.
